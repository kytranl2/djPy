# djPy
Create a Virtual Environment: 
```python3 -m venv venv```
Activate the Virtual Environment:
* For Unix or MacOS:

    ```source venv/bin/activate```
* For Windows, use:
```venv\Scripts\activate```


Start application:
```
python manage.py runserver 8080

python manage.py runserver 0.0.0.0:8000
```
